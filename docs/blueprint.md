# **App Name**: Cozy Commerce

## Core Features:

- Product Catalog: Browse home decor items by category with filtering and sorting options.
- Product Details: View detailed information, images, and customer reviews for each product.
- Shopping Cart: Add, remove, and update items in a persistent shopping cart.
- Simplified Checkout: Secure and streamlined checkout process with guest checkout option.
- User Login: Authentication system for customers to manage their profiles and order history.
- Seller Registration: Page with information about how to become a seller on the platform.
- Personalized Recommendations: Suggest relevant products to customers based on their browsing history.

## Style Guidelines:

- Primary color: Terracotta (#E07A5F) for warmth and earthiness.
- Background color: Desaturated terracotta (#F2DED8) as a soft backdrop.
- Accent color: Cream (#F8F2ED) to create highlights and provide contrast.
- Headline font: 'Playfair', a modern sans-serif with an elegant, fashionable feel.
- Body font: 'PT Sans', a humanist sans-serif suitable for comfortable reading.
- Use simple, line-based icons in terracotta color for product categories and navigation.
- Minimalist layout with generous white space to create a clean and airy feel.
- Subtle fade-in animations for product images and transitions between pages.
